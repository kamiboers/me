FitBeat
A fitness music app consuming Fitbit and Spotify APIs. FitBeat pulls in daily heart rate data, displays it on a dashboard and allows the user to create and populate Spotify playlists whose BPM falls within the ranges of Fitbit's exercise heart rate categories. Uses Ruby on Rails, OAuth2 authorization, jQuery, RSpec, and VCR to test API interaction.

IdeaBox
Single-page app to store and rate 'ideas'. Uses Rails to manage and serve data and JavaScript/jQuery for client-side interactions. All CRUD functionality is initiated via jQuery AJAX calls to the Rails server. Ideas can be edited inline, filtered by tags, and searched dynamically without page reload. jQuery is responsible for all DOM manipulation.
